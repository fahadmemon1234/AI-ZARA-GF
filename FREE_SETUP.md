# 🚀 Zara AI - FREE Setup Guide

## ⚡ QUICK START (5 Minutes!)

### Step 1: Get FREE Groq API Key

**Groq is 100% FREE, FAST, and NO credit card needed!**

1. Visit: **https://console.groq.com**
2. Click **"Sign Up"** (use Google account - easiest!)
3. Verify email
4. Click **"API Keys"** in left menu
5. Click **"Create API Key"**
6. Name it: "Zara AI"
7. **COPY THE KEY** (starts with `gsk_`)

### Step 2: Add Key to .env

Open: `D:\Fahad Project\AI-Driven\zara-ai\.env`

Replace this line:
```env
GROQ_API_KEY=gsk_your-groq-key-here
```

With your actual key:
```env
GROQ_API_KEY=gsk_XXXXXXXXXXXXXXXXXXXXXXXXXXXXXXXXXXXX
```

**Save the file!**

### Step 3: Install Groq (One Time)

```bash
pip install groq
```

### Step 4: Restart Server

```bash
# Close current server (Ctrl+C if running)
python main.py
```

### Step 5: Open Browser

```
http://localhost:8000
```

### Step 6: Chat with Zara! 💕

Type: "Hey Zara" or "I love you"

## 🐛 Troubleshooting

### "No API key" error
**Solution:** Make sure you added Groq key to `.env` file

### "Module not found" error
**Solution:** Run `pip install groq`

### Server not starting
**Solution:** 
```bash
# Kill any running Python processes
taskkill /F /IM python.exe

# Restart
python main.py
```

### Zara not responding
**Solution:** 
1. Check Groq API key is valid
2. Check server is running (look for "Zara is ready!")
3. Refresh browser page
